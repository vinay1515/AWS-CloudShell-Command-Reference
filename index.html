What is AWS CloudShell?
AWS CloudShell is a browser-based shell that makes it easy to securely manage, explore, and interact with your AWS resources. CloudShell is pre-authenticated with your console credentials, providing a seamless transition from the graphical console to the command line interface without requiring local credential configuration.
Security & Persistence
  • Pre-authenticated: Inherits the IAM permissions of your active AWS Management Console session automatically.
  • Persistent Storage: Provides 1 GB of persistent storage per AWS Region within your home directory ($HOME), ensuring scripts and config files remain across sessions.
  • Pre-installed Tools: Comes equipped with the AWS CLI, Python, Node.js, and bash utilities out of the box.

Best Practice

Always verify your active IAM identity using aws sts get-caller-identity before executing destructive commands to prevent accidental cross-account modifications.
